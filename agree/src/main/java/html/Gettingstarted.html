Getting started

Loading schema/STEP file

Load a schema from "File"->"Load schema" or open an existing STEP file, "File" -> "Import STEP". AP-schemas are parsed in advance and thus faster to load.

Giz files

AGREE files are saved with the file extension *.giz. Thus files created in Eurosteps Graphical Instance can be opened in AGREE.

Creating entity instances

Entity instances are created by "drag-and-drop" from the entities in the loaded schema.
