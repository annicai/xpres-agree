Controls

Zoom

Mouse-scroll in/out.
CTRL+I / CTRL+O.

Change location on the canvas

Mouse scroll while pressed.
Key-arrows up/down/left/right.

Resize instance box

Double-click on the box. Hold down and drag the mouse.

Fit view

CTRL+F. Makes the instance diagram fit on the visible canvas.

Save

CTRL+S.

Undo

CTRL+Z.

Redo

CTRL+Y.
