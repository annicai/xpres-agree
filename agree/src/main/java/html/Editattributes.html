Edit attributes

String/Integer/Double

To edit a value attribute, double-click on the attribute in the instance box and type a new value.

Entities

Click on the corresponding button in the instance box. Drag existing instances from the left table to the right (green) to add the entity to the attribute and from the right to the left to remove it.
New entities are created by dragging them from the entity tree in the "New instances"-tab.
The shown existing instance matches can be filtered by choosing an entity class in the "Filter"-tab. The instances can also be filtered by their (string) attribute values.

Non-entity aggregates

Non-entity aggregates are typed in the instance boxes. Aggregates are grouped by parentheses and separated by commas. Eg: (3,4,5) or ((3,4,5),(6,7,8))

Enumeration / Boolean / Logical

Double-click on the attribute in the instance box. Scroll through the options with the mouse scroll button. Double-click to set the attribute.
Alternatively right-click on the instance box and choose "Edit attribute" -> The attribute of choice -> Select a value.

Selects

Non-entity select attributes are set like Enumerations, but when the select type has been chosen the select value is entered in a pop-up window.
